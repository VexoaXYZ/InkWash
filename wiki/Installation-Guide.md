# üì• Installation Guide

This guide will help you download and install InkWash on your computer. It's super easy - just 3 steps!

## üéØ Step 1: Download InkWash

1. Go to the [Releases page](https://github.com/VexoaXYZ/InkWash/releases/latest)
2. Look for the **"Assets"** section
3. Click on `inkwash-windows-amd64.zip` to download it

![Download Screenshot](https://via.placeholder.com/600x200/7c3aed/ffffff?text=Download+from+Releases)

**Tip:** Make sure to download the latest version for the newest features and bug fixes!

## üóÇÔ∏è Step 2: Extract the Files

1. Find the ZIP file you downloaded (usually in your Downloads folder)
2. Right-click on `inkwash-windows-amd64.zip`
3. Click **"Extract All..."**
4. Choose where you want to extract it (Desktop is fine for beginners!)
5. Click **"Extract"**

![Extract Screenshot](https://via.placeholder.com/600x200/7c3aed/ffffff?text=Extract+All)

## üéÆ Step 3: Run InkWash

1. Open the folder where you extracted the files
2. Double-click on `inkwash.exe`
3. You're done! InkWash is now running!

![Run Screenshot](https://via.placeholder.com/600x200/7c3aed/ffffff?text=Double+Click+inkwash.exe)

## ‚öôÔ∏è Optional: Add to PATH (Advanced)

Adding InkWash to your PATH lets you run it from anywhere in Command Prompt. This is optional but convenient!

### Windows 10/11:

1. Press `Win + R` to open Run dialog
2. Type `sysdm.cpl` and press Enter
3. Go to the **"Advanced"** tab
4. Click **"Environment Variables"**
5. Under **"System Variables"**, find **"Path"**
6. Click **"Edit"** ‚Üí **"New"**
7. Paste the folder path where `inkwash.exe` is located
8. Click OK on everything
9. **Restart Command Prompt**

Now you can use `inkwash` from anywhere!

## ‚úÖ Verify Installation

To make sure InkWash is installed correctly:

1. Open Command Prompt (Press `Win + R`, type `cmd`, press Enter)
2. If you added to PATH: Type `inkwash --version`
3. If not in PATH: Navigate to InkWash folder and type `inkwash.exe --version`

You should see the version number!

## üîÑ Updating InkWash

To update to a new version:

1. Download the new version from [Releases](https://github.com/VexoaXYZ/InkWash/releases/latest)
2. Extract it to the same folder (overwrite old files)
3. Done! Your servers and settings are safe.

**Note:** Your servers and license keys are stored separately, so updating won't delete them!

## ‚ùì Troubleshooting

### "Windows protected your PC" message

This is normal for new downloads! Click **"More info"** ‚Üí **"Run anyway"**

### Can't extract the ZIP

Make sure you have:
- Windows 7 or higher
- Enough disk space (InkWash is only ~15MB)
- File isn't corrupted (try downloading again)

### InkWash won't run

Make sure you:
- Extracted the ZIP (don't run from inside the ZIP!)
- Have Windows 7 or higher
- Downloaded the Windows version (inkwash-windows-amd64.zip)
